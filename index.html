Мой учебный процесс

git

  • git init
  • git status
  • git add hello.txt
  • git add -A (чтобы добавить все из папки)
  • git commit -m "Initial commit."
  • git branch -M main
  • git remote add origin origin это наше произвольное название https://......(Подключение к удаленному репозиторию)
  • git push -u origin main (переслать наш локальный коммит на сервер)
