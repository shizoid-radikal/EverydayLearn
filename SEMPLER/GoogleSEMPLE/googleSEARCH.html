Регулярные выражения

  • Точное совпадение ковычки: "Взлом"
  • OR- с большой буквы: "Взлом" OR "wi-fi"
  • исключить слово "-" : "Взлом" -wi-fi
  • неизвестные буквы: **link
  • ОПЕРАТОРЫ:
    • site:habr.com "взлом"
    • intitle: поиск по заголовку
    • inurl: поиск по url
    • cache:habr.com => покажет снимок крайней индексации
    • link:habr.com => поиск ссылающихся на этот адрес
    • alltitle:"взлом wi-fi" => страницы со всей фразой
    • allintext: "взлом wi-fi" => выведет страницу с искомой фразой
    • "Взлом" filetype: pdf
  • ПРИМЕРЫ:
    • intitle: "index of/" Parent Directory
    • intitle: "index of/mail" site:...
    • intitle: "index of/admin" site:...
