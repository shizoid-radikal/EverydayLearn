Знания из игры выше

  • git commit
  • git branch "<"name>"
  • git merge bugFix // мы в Main
  • git rebase main // установит копию под main c '
  • git checkout C1 // (hash) сумма // HEAD
  • Перемещение:
  • git checkout main^ // на одну ветку выше
  • git checkout main^2 // выберет вторую (не основную) ветку
  • git checkout HEAD^
  • git checkout HEAD~3 // на 3 вверх относительно HEAD
  • git branch -f main HEAD~3 // main переместится на 3ю ветку от HEAD
  • //-f forced
  • Отмена изменений
  • git reset HEAD~1 // удаляет последующие,остается на HEAD~1
  • git revert HEAD // чтобы поделиться,отмененными изменениями
  • git cherry-pick C2 C4 // типо rebase - мы в main
  • git gui
  • git gitk
  • git rebase -i HEAD~3
  • git commit --amend //Изменение последнего коммита,забыл написать -m""
  • git tag V1 C1 // установит TAG V1 для С1
  • git describe "<"ref"> // ref-это конкретный commit//git describe main//
  • // укажет расстояние до Тега // tag_расстояние_hash
  • git checkout HEAD~^2~2
